Prevents access to assets of inactive plugins/themes via .htaccess rules

== Description ==

This plugin prevents access to assets of inactive plugins and themes. Rules are enforced via `.htaccess` files, updated each time the active status of a plugin or theme changes.

[GitHub](https://github.com/GoWP/gowp-cold-storage)
